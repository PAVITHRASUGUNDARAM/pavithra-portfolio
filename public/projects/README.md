# Project Screenshots

Add your screenshots here. The portfolio loads images from these folders.

## Folder structure

```
public/projects/
  diabetes/
    screen1.jpeg … screen7.jpeg
  chatbot/
    screen1.jpeg … screen4.jpeg
  jewelry/
    screen1.png
    screen2.png
    screen3.png
  amplify-cognito/
    screen1.png
    screen2.png
  spotify/
    screen1.png
    screen2.png
```

## Notes

- Use **exact file names** (`screen1.png` or `screen1.jpg`, `screen2`, …).
- Supported: `.png`, `.jpg`, `.jpeg`, `.webp` — the site tries these automatically if one is missing.
- Example: AWS Cognito uses `screen1.jpg` … `screen6.jpg` in `amplify-cognito/`.
- AWS Quiz was removed from the showcase (extra duplicate screenshots only).
- AWS Cognito project uses `public/projects/amplify-cognito/`.
